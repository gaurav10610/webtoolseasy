How to use screen recorder

      

        

          1. To include mic audio & camera video in the recorded video, use
          checkboxes
        

        
2. Start screen recording using start recording button

        
3. When asked for mic & camera permission then provide the same

        

          4. Camera video if included will appear at bottom right corner in the
          recorded video in 150X150 px box
        

        

          5. Screen recording can be stopped via native stop recording button or
          use stop recording button on the screen (button will appear once
          recording starts)
        

        

          6. Once recording is stopped then recorded video will be downloaded
          automatically
        

      

      
Features

      

        
1. Screen recording is 100% free

        
2. Unlimited screen recording in HD format

        
3. Record screen with camera video & mic audio

        
4. No software installation is needed to record screen

        

          5. Screen recording is 100% secure as whole processing is being done
          at client side only (in host browser)
