How to compare text

      

        
1. Paste original text in text 1 area

        
2. Paste target text in text 2 area

        

          3. Difference between both the texts will be evaluated automatically
          and will be reflected in difference area
        

        

          4. Default text comparion type is charcter by charcter, which can be
          changed via comparison type dropdown to any of the available
          comparison type like word by word or line by line
        

        
5. To ignore case while comparing, use the corresponding checkbox

        

          6. To ignore leading & trailing whitespace while comparing, use the
          corresponding checkbox
        

        

          7. For swapping text 1 & text 2 with each other use the switch texts
          button
        

        

          8. After making any of the above change difference between both the
          texts will be evaluated automatically
